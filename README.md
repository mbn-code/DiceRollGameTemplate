## Introduction
This simple dice roll game allows you to roll a virtual six-sided die and compete against an AI opponent. The goal is to roll higher than the AI and win as many rounds as you can.

## Features
- Randomized dice rolls for both player and AI.
- Simple scoring system to track wins and losses.
- Customizable AI difficulty.
- Rounds system for extended gameplay.

## How to Play
1. Clone the repository to your local machine.
2. Run the game script in your preferred Python environment.
3. Follow the on-screen instructions to roll the dice and compete against the AI.
4. Choose to continue playing or exit after each round.

## Gameplay
- Type "roll" to roll the dice.
- Type "exit" to end the game.
- The AI's roll is compared to yours, and the winner is determined.
- The game continues until you choose to exit.

## Customization
Feel free to customize the game by modifying the script:
- Adjust the sleep times for a faster or slower pace.
- Experiment with different difficulty levels for the AI.
- Add graphics or enhance the user interface.

## Scoring
The game keeps track of your wins and losses. The score is displayed after each round, allowing you to track your progress.

## Requirements
- Python 3.x

## Installation
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your-username/dice-roll-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dice-roll-game
   ```

## Usage
Run the game script using your preferred Python interpreter:
```bash
python dice_roll_game.py
```

Follow the on-screen instructions to play the game.
